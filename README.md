# Cundina App Starter (Next.js + Supabase)

**Brand:** CUNDINA — Circle of Savings • A modern way to save together.
**Date:** 2025-10-29

Quick start:
1) Create a Supabase project, copy URL & keys.
2) Copy `.env.example` into `web/.env.local` and fill values.
3) Run migrations (in Supabase SQL editor) in order: 000, 002, 003.
4) `cd web && npm install && npm run dev` → open http://localhost:3000
5) Deploy to Vercel and add the same env vars.
